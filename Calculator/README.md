# Simple Calculator 

A basic, command-line arithmetic calculator built in Python. This project was created to practice fundamental Python concepts like continuous loops, conditionals, and basic error handling.

## Features 

* **Basic Arithmetic:** Performs Addition (`+`), Subtraction (`-`), Multiplication (`*`), and Division (`/`).
* **Continuous Execution:** Runs in a continuous loop so you can perform multiple calculations without having to restart the script.
* **Error Handling:** Safely handles invalid inputs (like typing letters instead of numbers) using `try/except` and prevents program crashes from division by zero.
* **Graceful Exit:** Easily shut down the program at any time by typing `q`.

## Concepts Practiced 

* `while` loops for program state management.
* `if/elif/else` conditional logic.
* Exception handling (`ValueError`).
* User input and type casting (`float()`).
* String formatting using f-strings.
